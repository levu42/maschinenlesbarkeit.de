Über das Projekt


Die Forderung nach Maschinenlesbarkeit ist ein wichtiger Teil von OpenData. Diese Website bietet einen Überblick sowohl über bestehende APIs als auch über selbst gehostede APIs für Daten ohne Möglichkeit des Zugriffs über eine API.

About the project


Maschinenlesbarkeit is German for machine readability which is an important part of OpenData. The aim of this website is to provide an overview of existing APIs as well as self hosted APIs for services without an API.
